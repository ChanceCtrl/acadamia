#motorsports 
# Enter/Leave
- NEVER be in the machine shop alone when machines are running.
	- Always have at least 1+ person with you.
- NEVER leave the machine shop when machines are running
	- Safety glasses MUST be warn at all times.

# Clothing rules
- No loose clothing including ties, long jewelry, loose necklass, floppy sleeves etc
- Long hair must be tied

# Cleaning rules
- Keep you work area clean
- Do NOT put tool/mats on top of the machine you are using
- Put tools away when you are done using them
- NEVER remove metal chips, turnings, or shavings, with your hands
- Report ANY damaged safety guards, machine and tools

# General Shop rules
- Music is allowed if on a speaker & quiet enough to still hear others speaking
- NO headphones/earbuds
- NO wood is allowed in the shop
- Only work with tools and machines that you have taught to use
- NEVER push a cutter towards any part of your body
- Always secure the work piece with clamps or a vise
- IF you think something is wrong, it probably is, stop and ask for help before hurting yourself or the machine
- No running or horseplay in the machine shop
- No eating in the shop area
